# 👁️ Advanced ResNet50 Autoencoder Framework for Multi-Class Eye Disease Classification

A deep learning-based approach that combines a **Residual Autoencoder** for feature extraction with **ResNet50** for accurate classification of retinal diseases such as Cataract, Diabetic Retinopathy, and Glaucoma.

---

## 📌 Overview
This project presents a **hybrid deep learning framework** for automated multi-class eye disease classification using retinal fundus images.

It integrates:
- Residual Autoencoder for feature extraction  
- ResNet50 model for classification  

### 🎯 Objective
To assist in the **early detection** of eye diseases such as:
- Cataract  
- Diabetic Retinopathy  
- Glaucoma  

---

## 🚀 Features
- 🔍 Automated retinal image analysis  
- 🧠 Hybrid architecture (Autoencoder + ResNet50)  
- 📊 Multi-class classification  
- ⚡ Improved feature representation  
- 🏥 Supports early diagnosis and medical assistance  

---

## 🏗️ Project Architecture

### 1. Image Acquisition
- Collection of retinal fundus images from datasets  

### 2. Image Preprocessing
- Resizing images  
- Normalization (if applied)  

### 3. Feature Extraction (Autoencoder)
- Learns compressed and meaningful image representations  
- Reduces noise and enhances important features  

### 4. Classification (ResNet50)
- Pre-trained ResNet50 used for classification  
- Fine-tuned for eye disease detection
